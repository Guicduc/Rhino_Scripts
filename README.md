# Rhino Scripts

## CreateBomCommand

Este repositório contém um esqueleto de comando RhinoPython.

### Como executar no Rhino

1. Abra o Rhino.
2. Vá em **Tools > PythonScript > Run**.
3. Selecione o arquivo `src/create_bom_command.py`.
4. Execute o comando `CreateBomCommand()` no console Python ou deixe o script
   rodar direto.

O comando solicita a seleção de objetos e imprime a quantidade selecionada no
console do Rhino.
